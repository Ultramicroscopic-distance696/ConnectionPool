ConnectionPool helps Apple devices connect to each other over a local network or through a remote relay. It is built for iPhone, iPad, and Mac.

ConnectionPool uses two connection types:

- Local mesh links with MultipeerConnectivity

In simple terms, the app can connect devices that are near each other or far apart.

This helps with:

- Shared apps
- Multiplayer tools
- Private messaging systems
- Cross-device data exchange
- Local testing between devices

## 🔍 Troubleshooting

### App does not open

- Check that the download finished
- Make sure you extracted the ZIP file first
- Try running the file as administrator
- Restart Windows and open the app again

### Windows blocks the app

- Right-click the file
- Select Properties
- If you see an Unblock option, enable it
- Try opening the app again

### No devices appear

- Check that both devices are on the same network
- Make sure local network access is allowed
- Keep the app open on both devices
- Turn off VPN tools that may block discovery

### Connection fails over relay

- Check your internet connection
- Confirm the relay server is online
- Verify the relay address in the app
- Try closing and reopening the app

## ⚙️ Network Terms in Plain English

- **Peer**: another device you connect to
- **Mesh**: a group of devices that can connect through each other
- **Relay**: a helper server that passes data between devices
- **Encryption**: a way to keep data private
- **Local network**: devices on the same Wi-Fi or router
